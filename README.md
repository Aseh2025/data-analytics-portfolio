# Sales Performance Dashboard (Excel)

## Project Overview
This project analyzes sales data to identify trends, profitability drivers, and regional performance using Microsoft Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Excel Charts & Slicers

## Key Features
- KPI cards (Sales, Profit, Margin)
- Category & Sub-category analysis
- Regional performance comparison
- Time-series sales trends
- Interactive slicers

## Key Insights
- Technology is the most profitable category.
- West region leads in profit.
- Clear seasonal sales patterns exist.
- Copiers and Phones drive profitability.

## Recommendations
- Focus growth strategy on Technology.
- Strengthen West region operations.
- Improve pricing for low-margin categories.
